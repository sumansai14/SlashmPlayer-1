Welcome to the official website of SlashmPlayer!

Why SlashmPlayer?

  • Got bored using all those players that seem like they are not made for you?
  • Got bored with having players with junk features you would never use?

Ever thought how simple a media player might be if it only had those features you always use??

Well, we thought it out for you already... Slashmplayer is going to be that one simple media player made just for you...

Go ahead, download and try it to experience it!
